You know what a Sweeps is. This app has the basic workings for running a sweep, although it probably misses the most important part, the collecting and distribution of money.

It's a javascript app, built with AngularJS framework, with help from Bootstrap and HTML5 Boilerplate. The backend is a MySQL database, and there is some PHP and SQL to glue the thing together.

This is version 1 of the app and I've only implemented the the bare bones required to get a sweep working. I'm not sure I'd even use it myself at the moment. Nevertheless, it was a fun experiment for me and it's here to play around with.

Code available from Github.
